# FISG Trading Hub Prototype (React Native)

This is a clickable mobile UI prototype for the FISG Trading Hub MVP + growth plan.

## Run

```bash
npm install
npm run dev
```

> Expo shortcut: use `npm run start` if you prefer Expo default.

## Notes

- React Native (Expo) prototype with mock data only.
- Hidden Dev Panel: tap the small dot at the top-right to toggle status simulations.
- Bottom tabs: Home, Market, Trade, Wallet, Rewards, News, Settings, More (Legacy placeholder).
- i18n: switch language in Settings (EN/TH/CN).

## Key Files

- `App.tsx` – App shell, navigation, toast, dev panel
- `src/screens/*` – screen components
- `src/components/*` – reusable UI components
- `src/state/*` – app state + i18n
- `screens-and-flows.md` – sitemap, flows, states, edge cases
